# 🎯 Bank Client Target Response Prediction

Построение модели машинного обучения для предсказания отклика клиента банка 
на маркетинговое предложение и формирование портрета целевой аудитории.

---

## 📋 Описание проекта

Проведён полный цикл анализа данных клиентов банка для выявления ключевых 
характеристик клиентов, склонных к положительному отклику на рекламную кампанию.

**Цель:** построить прогностическую модель, которая позволит банку точечно 
обращаться только к тем клиентам, которые с наибольшей вероятностью откликнутся 
на предложение, тем самым сократив маркетинговые расходы.

## 🔑 Ключевые этапы работы

### 1️⃣ Предобработка данных
- Переименование столбцов, исправление типов данных
- Заполнение пропусков: медиана для числовых признаков, «не указано» для категориальных
- Выявление и обработка аномалий (стаж работы, доход, сумма кредита) методом IQR

### 2️⃣ Генерация новых признаков
- Временные признаки из даты кредита: год, месяц, день недели
- Признаки территориальной стабильности: совпадение адресов регистрации, 
  фактического проживания и почтового адреса

### 3️⃣ Исследовательский анализ (EDA)
- Динамика выдачи кредитов по годам, месяцам и дням недели
- Анализ числовых и категориальных признаков в разрезе целевой переменной
- Корреляционная матрица и топ-10 факторов влияния на отклик

### 4️⃣ Моделирование и сравнение моделей

| Модель | ROC-AUC | F1 (класс 1) | Recall (класс 1) |
|---|---|---|---|
| Логистическая регрессия | 0.64 | 0.25 | 0.25 |
| Random Forest + SMOTE | 0.60 | 0.03 | 0.01 |
| XGBoost + SMOTE | 0.60 | 0.23 | 0.51 |
| **XGBoost (tuned, порог 0.54)** | **0.64** | **0.26** | **0.45** |

✅ Лучшая модель — **XGBoost** с подобранными гиперпараметрами 
и оптимальным порогом классификации 0.54

## 📊 Портреты клиентов

| Характеристика | Склонный к отклику | НЕ склонный к отклику |
|---|---|---|
| Возраст | 30–45 лет | 45–60 лет |
| Занятость | Работающий | Не работает / пенсионер |
| Личный доход | Выше среднего | Ниже среднего |
| Просрочки | Отсутствуют | Есть |
| Адреса | Совпадают | Не совпадают |
| Кол-во детей | 1–2 | 3 и более |

---

## 💡 Ключевые инсайты

- **Сильные предикторы:** возраст, личный доход и территориальная стабильность клиента
- **Финансовая дисциплина:** клиенты без просрочек значительно чаще откликаются
- **Стабильность:** совпадение адресов регистрации и проживания — важный сигнал для скоринга

---

## 📌 Рекомендации для бизнеса

- Сфокусировать кампании на клиентах **30–45 лет** с доходом выше среднего
- Исключить из таргетинга клиентов с историей просрочек
- Использовать модель как скоринговый инструмент: обращаться только 
  к клиентам с вероятностью отклика **выше 0.54**
- Ожидаемый эффект: сокращение маркетингового бюджета при охвате **45% целевой аудитории**
